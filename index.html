01 — Systems Engineering
ANNE — Approximate Nearest Neighbor Engine
99.6% recall@10 100× vs brute-force 4.9× SIMD speedup 1M SIFT vectors
  • Implemented HNSW graph algorithm from scratch in C++17; indexed 1M SIFT vectors achieving 99.6% recall@10 — within margin of production libraries like Faiss on the standard SIFT1M benchmark
  • Hand-wrote AVX2 SIMD distance kernels (_mm256_fmadd_ps) processing 8 floats/instruction; 4.9× faster than scalar baseline on the search hot path
  • Delivered 100× speedup over brute-force at 94% recall via cache-efficient flat vector storage and O(1) generation-based visited tracking
  • Shipped with 45 Google Test unit tests, Google Benchmark microbenchmarks, and GitHub Actions CI pipeline
